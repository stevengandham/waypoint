A full-featured, glassmorphism-styled landing page for Waypoint — Beachpoint Church's 25–32 community.

Vite + React Tailwind CSS v4 Google Sheets Integration Scroll Spy Glassmorphism Responsive Mobile Nav

6-Phase Build

1

Scaffold

Vite + React + Tailwind v4 setup. Folder structure, content.js data layer, gradient orb CSS.

2

UI Kit

Reusable primitives: Button, Card, TabButton (nav pill), Toast notification.

3

Shell

Navbar (logo + desktop/mobile nav + scroll-aware glass), Footer, PageShell layout wrapper.

4

Sections

All 8 sections: Hero, About, Vision, Values, Contact, Socials, Gallery, InterestForm.

5

Form + Sheets

InterestForm with validation → useGoogleSheet hook → Google Apps Script Web App.

6

Polish

Scroll spy, stagger animations, logo integration, mobile responsive, build optimization.

Key Features

Mobile Responsive

📊
Google Sheets Integration

Logo Integration Details

The logo file waypoint.jpg was placed as public/images/logo.jpg and is referenced via the standard Vite public directory path.

// Direct path from Vite's public/ folder
src="/images/logo.jpg"
// Styled with Tailwind: 40×40px, rounded, object-fit cover
className="h-10 w-10 rounded-lg object-cover"

Appears in Navbar.jsx (line 36). Rendered on all pages via the shared layout shell.
